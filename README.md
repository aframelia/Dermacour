# Demarco Couriers – Static Website

A modern, responsive static website for **Demarco Couriers**, a London-based same-day delivery and logistics service.

This project showcases a clean UI focused on clarity, trust, and fast customer conversion — highlighting services, pricing, and easy contact options.

---

## Features

-  **Responsive Design** – Optimised for mobile, tablet, and desktop  
-  **Fast Loading** – Lightweight static HTML & CSS  
-  **Quick Contact Options**
  - Click-to-call phone link  
  - WhatsApp direct messaging  
-  **Smooth Navigation** – Scroll-based sections for easy browsing  
-  **Service Showcase** – Clear breakdown of courier offerings  
-  **Pricing & Offers Section** – Loyalty deals and business options  
-  **Trust Signals** – Insurance, reliability, and service guarantees  

---

##  Tech Stack

- HTML5  
- CSS3  
- No frameworks or libraries – pure static implementation
